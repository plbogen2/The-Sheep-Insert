# The Sheep: Apothecary Victorian Insert (Modular)

This is a complete, custom 3D-printable modular board game insert for "The Sheep" (by Zero Strategy Games). It replaces the standard cardboard divider with a set of Victorian-inspired "Apothecary" deck boxes and token bins, designed specifically to fit a 9 1/4" (234.95mm) box interior. This insert dramatically speeds up your game's setup and teardown times, keeping everything securely in place.

The design features a custom-engineered Victorian Ironwork lid pattern. When printed on a Satin PEI sheet, the lids take on a beautiful matte, cast-iron texture that perfectly suits the comedy-horror theme of the game.

## 🏛 The "Apothecary" Aesthetic

* **Wrought Iron Lids:** Ornate circular "Rose Window" lattice that allows you to see card art/tokens through the lid.
* **Debossed Labels:** Clean, bold typography (using Alfa Slab One) that matches the rulebook nouns.
* **Bottom-Up Printing:** Optimized for printing on textured sheets to give the "ironwork" a high-end, non-plastic finish.

## 📦 Included Components

This set contains 8 modular containers designed to tile perfectly within the original game box. *(Note: Box dimensions and labels have been updated to support sleeved cards and match the latest component lists!)*

* **Deck Boxes (Poker Size):** 
  * `EQUIPMENT` Deck Box
  * `EVENTS` Deck Box
  * `CURSES` Deck Box
  * `RESOURCES` Deck Box
* **Deck Boxes (Tarot Size):**
  * `MUTATIONS` Deck Box
* **Trait Trays:** A dedicated 6-slot tray for tracking character traits (`MERC`, `DOC`, `SURV`, `SGT`, `MECH`, `SCOUT`).
* **Token & Tracker Bin:** Deep, filleted (rounded) storage compartments for easy scooping of `ESSENCE`, `LANTERNS`, `CORRUPTION`, `EXPERIENCE`, and `POLLUTION` tokens.
* **Sheep Bag Tray:** A large, open-topped compartment specifically designed for the drawstring bag.

## 🖨 Recommended Print Settings

* **Printer:** Optimized for MK3.9 / MK4 (Load cell ensures perfect first-layer adhesion for the fine lattice).
* **Sheet:** Satin PEI (Highly recommended for the cast-iron look).
* **Material:** Black PLA (Standard) or Charcoal/Gunmetal Grey.
* **Layer Height:** 0.2mm (to ensure the 1.2mm lid pattern is a sturdy 6 layers).
* **Lid Orientation:** Print Bottom-Down (Face down) to get the sheet texture on the decorative side of the lid.
* **Supports:** None required.

## 🛠 Technical Details

* **Total Footprint:** ~235mm x 320mm.
* **Lid Thickness:** 1.2mm (Structural Ironwork).
* **Card Compatibility:** Supports sleeved cards (checked against standard poker/tarot sleeves).
